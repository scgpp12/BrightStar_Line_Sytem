# 人事アシスタント（HR向け）利用マニュアル

**BrightStar 統合 LINE アシスタント — 人事チャネル**
対象：人事（HR）担当者 / 版本：v1.1 / 最終更新：2026-06-26

> 本チャネルは **人事担当（HR）専用** です。社員からの勤怠・通勤費の提出は「BrightStar 社員アシスタント」が受け付けます。
> **【2026-06-19 変更】** 勤怠・通勤費の **提出状況確認 / 未提出抽出 / 催促 / 一括DL は「総務アシスタント」へ移管**しました（[05 総務 利用マニュアル](05_総務_利用マニュアル.md)）。本チャネルは **社員名簿の管理（CRUD）** と **メール校正ツール** に専念します。
> 中文：本窗口仅限**人事(HR)**使用。考勤·通勤费的**提交状况/未提交/催促/打包下载已移到「総務アシスタント」**；本窗口现负责**员工名簿管理**与**邮件校正**。

---

## 1. はじめに（毎日の本人確認）

人事情報を扱うため、**毎日（東京時間）最初の利用時に本人確認**が必要です。

### 1-1. 初回登録（最初の1回だけ）
```
人事部 鈴木
```
- 社員名簿と照合し、**役割が HR（人事）** の本人のみ登録できます。
- 同姓同名は社員番号を付与：`人事部 鈴木 E001`
- 一度登録すると LINE アカウントと社員番号が**専用に紐付き**ます。

### 1-2. 毎日の認証
2日目以降は **「認証」** ワンタップ（またはメニューの「本日認証」）でその日の確認が完了します。

### 1-3. 言語の選択
認証後、最初に **「日本語」／「中文」**（既定：日本語）。後から「日本語」「中文」で切替可。

### 1-4. 登録のやり直し
**「登録解除」** で紐付けを解除し、別人で登録し直せます。

## 3. 操作の詳細

### 3-0. 提出管理（一覧 / 未提出 / 催促 / 一括DL）は総務へ移管
勤怠・通勤費の **提出状況確認・未提出抽出・催促・一括ダウンロード** は「総務アシスタント」で行います。
→ [05 総務 利用マニュアル](05_総務_利用マニュアル.md) を参照してください。

### 3-1. 社員名簿の管理（CRUD）
| 操作 | 送信例 | 内容 |
|------|--------|------|
| 一覧 | `名簿` | 全社員（ID／氏名／部署／役割／LINE紐付け状況） |
| 追加 | `社員追加 山田太郎 開発部 hr` | 新規社員を追加（役割省略時は一般社員） |
| 変更 | `社員変更 E003 部署 営業部
社員変更 E001 役割 hr,講師     ← 複数役割（カンマ区切り。hr/講師/営業/社員）
社員変更 E003 属性 外国籍      ← 部署の細分類（国籍・分類など任意）` | 氏名／部署／役割／別名を変更 |
| 別名設定 | `社員変更 E001 別名 孫成功,ソンセイコウ` | 認証で使える氏名の別表記（簡体/繁体/カナ/ハングル/ローマ字） |
| 削除 | `社員削除 E003` | 名簿から削除＋LINE紐付け解除 |

> **氏名の表記ゆれ対応**：認証の「所属部署 お名前」は、全角/半角・かな（ソン＝そん）・大小文字を自動で同一視します。さらに **別名（aliases）** を登録すれば、簡体字「孙成功」／繁体字「孫成功」／カナ「ソンセイコウ」など複数表記のどれでも本人確認できます。中国籍の社員などは別名にフリガナ・繁体字を入れておくと便利です。

- 役割（role）の値：`employee`（一般）／`hr`（人事）／`teacher`（講師）／`sales`（営業）／`soumu`（総務）。
- 名簿を変更すると、**他の HR 全員に変更通知**が自動配信されます（変更者と内容入り）。
- 一覧に **🚫ブロック中** が付く人は、その LINE アカウントが bot をブロック/削除しています。毎日 0:00 に自動点検し、使えるアシスタント全てで到達不可なら紐付け（LINE登録）が自動で解除されます（本人が再追加・再登録すれば元通り）。

### 3-2. メール校正ツール
```
メール校正
```
社内向けの **メール自動校正ツール**（社内ローカル AI）の URL を返します。ブラウザで開いて利用してください。

## 4. 提出テンプレート・提出管理について

社員が提出に使う **勤怠・通勤費のテンプレート** は社員アシスタント側から配布されます。人事チャネルは提出の**受領を行わず**、**社員名簿の管理**と**メール校正**に専念します。提出状況の確認・未提出抽出・催促・一括DL は **総務アシスタント**で行います（責務分離）。

### 参考：社員の提出内容の自動チェック
提出受領・管理は社員／総務アシスタントですが、人事でも内容を把握できるよう要点を記載します。

- **交通費経費申請表**：❌不可＝対象年月(B1)不一致／氏名(D6)不一致(簡繁OK)／出勤状態(D列)に未記入の日／⑤本人確認チェック未チェック。🔔リマインド＝責任者氏名(D7)未記入（定期券購入時必須）・成功時「定期券は写真添付必須」。
- **作業時間記録簿（勤怠）**：❌不可＝対象年月(B5)不一致／氏名(B3)不一致(簡繁OK)。⚠️警告＝休日勤務あり／日付が月末まで未充足。

> 詳細は [05 総務 利用マニュアル](05_総務_利用マニュアル.md) / [04 社員 利用マニュアル](04_社員_利用マニュアル.md) を参照。

## 5. よくある質問

**Q. 社員が送った Excel をこの窓口に転送できますか。**
できません。提出は社員アシスタントが受領します。人事チャネルは管理・集計用です。

**Q. 提出状況の確認や催促はどこでやりますか。**
「総務アシスタント」へ移管しました。[05 総務 利用マニュアル](05_総務_利用マニュアル.md) をご覧ください。

**Q. 名簿の役割を間違えました。**
`社員変更 社員番号 役割 hr` のように修正できます（総務担当は `soumu`）。
