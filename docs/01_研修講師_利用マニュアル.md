# 研修アシスタント（講師向け）利用マニュアル

**BrightStar 統合 LINE アシスタント — 研修チャネル**
対象：社内研修の **講師** / 版本：v1.0 / 最終更新：2026-06-19

> 本チャネルは **講師専用** です。受講（申込・キャンセル等）は「BrightStar 社員アシスタント」をご利用ください。
> 中文：本窗口仅限**讲师**使用；学员报名请用「社員アシスタント」。

---

## 1. はじめに（毎日の本人確認）

研修チャネルは情報保護のため、**毎日（東京時間）最初の利用時に本人確認**が必要です。

### 1-1. 初回登録（最初の1回だけ）
LINE で友だち追加したら、**「所属部署 お名前」**を送信します。

```
研修部 田中
```

- 社員名簿（花名册）と照合し、**講師(teacher)** 本人のみ登録できます。
- 同姓同名が複数いる場合は、社員番号も付けてください：`研修部 田中 E001`
- 名簿に無い場合は人事へご連絡ください。
- 一度登録すると、その LINE アカウントとあなたの社員番号が**専用に紐付き**ます（他人が同じ番号で登録することはできません）。

### 1-2. 毎日の認証（2日目以降）
翌日以降は、メニューの **「本日認証」** を押すか、**「認証」** と送信するだけでその日の確認が完了します。

### 1-3. 言語の選択
認証後、その日の最初に **「日本語」／「中文」** を選びます（既定は日本語）。選んだ言語で以後の応答が返ります。後から変えたいときは「日本語」「中文」と送れば切り替わります。

### 1-4. 登録のやり直し
別の人で登録し直す場合や誤登録のリセットは、**「登録解除」**と送信してください。紐付けが解除され、再度「所属部署 お名前」で登録できます。

## 2. コマンド一覧

すべて **1 通のメッセージで完結**します（多段の対話は不要）。フィールドの区切りは縦棒 **`|`** です。

| 操作 | 送信例 | 内容 |
|------|--------|------|
| ヘルプ表示 | `ヘルプ` / `講師ヘルプ` | 機能とキーワードをボタン付きで表示 |
| コース作成 | `建课 タイトル\|開講日時\|所要分\|定員\|紹介` | 新規コースを下書き(draft)で作成 |
| コース公開 | `发布 コース名` | Zoom 会議を作成し公開(published) |
| コース変更 | `改课 コース名\|項目=値;項目=値` | タイトル/定員/所要時間などを変更 |
| コース削除 | `删课 コース名` | コースを削除 |
| 受講者一覧 | `学员列表` | 全アクティブ受講者を一覧 |
| 申込名簿 | `名单 コース名` | 指定コースの申込者を一覧 |
| ランダム分組 | `分组 コース名\|组数=N`（または `\|每组=M`） | 申込者を自動でグループ分け |
| 本日認証 | `認証` / `本日認証` | その日の本人確認 |
| 登録解除 | `登録解除` | 別人で登録し直す |

> コース名は **部分一致**で検索します。複数ヒットした場合はより具体的な名称を送ってください。

## 3. 操作の詳細

### 3-1. コースを作成する（建课）
```
建课 Python入門|2026-06-20 14:00|90|20|未経験者向け
```
| 位置 | 項目 | 形式・規則 |
|------|------|-----------|
| 1 | タイトル | 必須。中日英・空白可 |
| 2 | 開講日時 | 必須。`YYYY-MM-DD HH:MM`（東京時間） |
| 3 | 所要時間 | 必須。分単位の整数 |
| 4 | 定員 | 必須。1 以上の整数 |
| 5 | 紹介 | 任意 |

成功すると **下書き（draft）** として保存されます。この時点ではまだ受講者には見えません。

### 3-2. コースを公開する（发布）
```
发布 Python入門
```
- Zoom 会議を作成し、コースを **公開（published）** にします。
- 返信に **主催者用リンク（あなた専用）** と **受講者用リンク** が表示されます。
- ※ 現在は Zoom 連携が **デモモード** の場合があり、その場合はサンプルのリンクが返ります。本番接続は Zoom 認証情報の設定後に自動で切り替わります（コード変更不要）。

### 3-3. コースを変更する（改课）
```
改课 Python入門|定員=30;所要時間=120
```
- 複数項目は **`;`** で区切ります。
- 変更可能：タイトル / 紹介 / 定員 / 所要時間 / 開講日時。

### 3-4. 受講者・申込の確認
- `学员列表` … 全受講者（重複排除）。
- `名单 コース名` … そのコースの申込者一覧。

### 3-5. ランダム分組（分组）
```
分组 Python入門|组数=4      ← 4グループに分ける
分组 Python入門|每组=5      ← 1グループ5名で分ける
```
申込者をランダムに割り当て、グループごとの名簿を返します。

### 3-6. 開講前リマインド（自動）
公開済みコースは、**開講の約1時間前**に申込者へ自動で LINE 通知（Zoom リンク付き）が届きます。講師側の操作は不要です。

## 4. よくある質問

**Q. 「受講ではありません」と出ます。**
あなたは講師として登録されていますが、受講の申込は社員アシスタントが担当です。研修チャネルは講師の管理操作専用です。

**Q. 認証が毎日求められます。**
情報保護のための仕様です。2日目以降は「認証」ワンタップで完了します。

**Q. コースが見つからないと言われます。**
コース名は部分一致です。短い特徴的な語で再送するか、`学员列表`/`名单` で正式名称を確認してください。

**Q. 別の講師アカウントに登録し直したい。**
`登録解除` → 改めて「所属部署 お名前」を送信してください。
