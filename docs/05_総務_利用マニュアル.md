# 総務アシスタント（総務向け）利用マニュアル

**BrightStar 統合 LINE アシスタント — 総務チャネル**
対象：総務担当者 / 版本：v1.1 / 最終更新：2026-06-26

> 本チャネルは **総務担当 専用** です。勤怠（勤務表）・通勤費（経費）の **提出状況の確認・未提出の抽出・催促・一括ダウンロード** を行います。
> 社員からの提出そのものは「BrightStar 社員アシスタント」が受け付けます。本チャネルは**受領は行わず**、管理・集計・督促に専念します（責務分離）。
> 中文：本窗口仅限**总务**使用；负责考勤(勤務表)·通勤费(经费)的**提交状况确认/未提交筛选/催促/打包下载**。员工提交请用「社員アシスタント」。

---

## 1. はじめに（毎日の本人確認）

人事・勤怠情報を扱うため、**毎日（東京時間）最初の利用時に本人確認**が必要です。

### 1-1. 初回登録（最初の1回だけ）
```
総務部 高橋 茉鈴
```
- 社員名簿と照合し、**役割が 総務（soumu）または 人事（hr）** の本人のみ登録できます。
- 同姓同名は社員番号を付与：`総務部 高橋 茉鈴 S001`
- 一度登録すると LINE アカウントと社員番号が**専用に紐付き**ます（1人＝1社員番号）。

### 1-2. 毎日の認証
2日目以降は **「認証」** ワンタップ（またはメニューの「本日認証」）でその日の確認が完了します。

### 1-3. 言語の選択
認証後、最初に **「日本語」／「中文」**（既定：日本語）。後から「日本語」「中文」で切替可。

### 1-4. 登録のやり直し
**「登録解除」** で紐付けを解除し、別人で登録し直せます。

## 2. コマンド一覧

| 操作 | 主なキーワード | 内容 |
|------|---------------|------|
| ヘルプ | `ヘルプ` / `菜单` | 機能とキーワードをボタン表示 |
| 提出状況の確認 | `一覧` / `提出状況` / `提交情况` | 全社員の提出状況（未提出を上に） |
| 未提出者の確認 | `未提出確認` / `未提出` / `未提交` | 未提出者だけを抽出 |
| 催促（リマインド） | `催促` / `リマインド` / `催办` | 未提出者へ LINE 通知を送信 |
| **催促予約** | `催促予約`（→日時ピッカー） | 指定日時に自動で催促（2026-07 追加） |
| 予約の確認/取消 | `予約確認` / `予約取消 <ID>` | 予約中の催促の一覧・取消 |
| **手動「済」** | 未提出確認の**ボタンをタップ** / `済 氏名` / `済解除 氏名` | メール等受領者を催促対象から除外 |
| 一括ダウンロード | `一括DL` / `打包下载` | 当月の全提出を ZIP でDL |
| **一斉送信** | `一斉送信`（管理者のみ） | 全社員へお知らせを配信（2026-07 追加） |
| 本日認証 | `認証` / `本日認証` | その日の本人確認 |
| 登録解除 | `登録解除` | 別人で認証し直す |

> 多くのコマンドは **月指定**が可能です（例：`未提出確認 202606`、`一括DL 202606`）。省略時は当月。
> 対象は **勤怠（勤務表）** と **通勤費（経費）** の2種類です。

## 3. 操作の詳細

### 3-1. 提出状況の確認
- `一覧` … 全員の提出マトリクス（✓＝提出済 / ✗＝未提出）。件数が多い場合は分割して送信します。
- `未提出確認` … 未提出者の氏名・部署・未提出の種別を一覧。
- いずれも末尾に `202606` のように月を付けると過去月を確認できます。

### 3-2. 催促（リマインド）
```
催促            ← 当月の未提出者へ
催促 202606     ← 月を指定
```
- 未提出者にだけ LINE 通知が届きます（提出済の人は自動で除外）。
- 通知文例：`【リマインド】{月} の {種別} が未提出です。期日までにご提出ください。`
- **毎月25日・28日 9:00（JST）に自動でも催促**が送られます（手動催促と併用）。
- 📌 リマインドは **「社員アシスタント」から社員へ届きます**（総務が催促を実行・確認し、通知は社員側のトークに表示）。総務担当には実行結果の件数のみ返ります。
- ⚠️ 社員が **「社員アシスタント」を友だち追加**していない／LINE未登録 の場合は通知が届きません（一覧には表示。総務担当が個別連絡）。

### 3-2b. 催促予約（日時を指定して自動催促／2026-07 追加）
```
催促予約        ← 日時ピッカーが開く
予約確認        ← 予約中の一覧（ID付き）
予約取消 a1b2c3 ← ID を指定して取消
```
1. メニューの **「催促予約」** をタップ → LINE の**日時選択画面**が開きます（明日 9:00 が初期値、60日先まで）。
2. 日時を選ぶと予約完了：`⏰ 催促を予約しました：2026-07-20 09:00（ID: a1b2c3）`
3. 予約時刻から **10分以内**に自動実行され、**その時点で未提出の人だけ**に届きます（提出済・手動「済」の人は対象外）。

### 3-2c. 手動「済」＝催促の対象から外す（2026-07 追加）
メール等で既に受け取っている人を、手動で「提出済み扱い」にできます（LINE未登録の社員にも使えます）。

**いちばん簡単な使い方（ボタン）**：
1. `未提出確認` を実行すると、一覧の下に **「済 ◯◯さん」ボタン**が人数分表示されます（先頭13名分）。
2. メール等で受領済みの人のボタンを**タップするだけ**で済になります。

コマンドでも指定できます（**氏名でも社員番号でも可**。一覧の行頭に社員番号が表示されます）：
```
済 山田太郎        ← 勤怠・経費の両方を済に
済 E003 勤怠      ← 社員番号＋種別指定
済解除 山田太郎    ← 取り消し
```
- 済にした人は `未提出確認`・`催促`・自動催促・催促予約の**すべてから外れます**。
- `一覧` では **✓(手)** と表示され、誰がいつ済にしたかも記録されます（監査用）。
- 14人目以降は「済 氏名」を送信して指定してください。

### 3-2d. 一斉送信（全社員へのお知らせ配信／2026-07 追加・管理者のみ）
メニューの **「一斉送信」** から、全社員への一斉連絡（お知らせ）を配信できます。

1. **「一斉送信」** をタップ → **「✏️ 入力フォームを開く」ボタン**が届きます
2. ボタンを押すと**入力フォーム（テキストボックス）**が開きます。内容を書いて **「📨 全社員に送信する」** をタップ（対象人数と確認ダイアログが表示されます）
3. 配信開始。完了後に LINE へ「成功◯名／失敗◯名」の結果が届きます
   - フォームを使わず、そのままトークに内容を送信してもOK（プレビュー→「送信する」で確定）。「キャンセル」で中止。
   - フォームのリンクは **30分有効**（切れたら「一斉送信」をもう一度）。

- 📌 お知らせは **「社員アシスタント」のトークから全社員へ届きます**（催促と同じ経路）。
- 対象は **LINE連携済みの全社員**（未連携・ブロック中は対象外。人数はプレビューに表示）。
- **権限**：花名册で「管理者」フラグのある人だけが使えます。付与は人事アシスタントで：
  ```
  社員変更 E001 管理者 true      ← 付与
  社員変更 E001 管理者 false     ← 解除
  ```

### 3-3. 一括ダウンロード
```
一括DL
```
- 当月の全提出ファイルを **ZIP** にまとめ、「全件ダウンロード」ボタンを返します。
- リンクは署名付きの短縮 URL で、一定時間で失効します。

## 4. 提出テンプレート・名簿・メール校正について

- 社員が提出に使う **勤怠・通勤費テンプレート** は社員アシスタント側から配布されます。
- **社員名簿の管理（追加/変更/削除）** と **メール校正ツール** は **人事アシスタント**側の機能です（総務チャネルにはありません）。

## 5. 参考：社員の提出内容の自動チェック

社員が提出すると、システムが内容を自動チェックします（提出の受領は「社員アシスタント」）。一覧で不備を確認・案内する際の参考にしてください。

**交通費経費**
- ❌ アップロード不可：対象年月(B1)が当月と不一致／氏名(D6)が登録名と不一致（簡体字・繁体字どちらでもOK）／**出勤状態(D列)に未記入の日がある**（その月の全日に 出勤/休み/在宅 等を記入必須）／**⑤本人確認のチェックが未チェック**
- 🔔 提出は可だがリマインド：責任者氏名(D7)が未記入・「（本社メンバーの名前）」のまま（定期券購入時は記入必須）／提出成功時「定期券は写真添付必須」

**勤務表（勤怠）**
- ❌ アップロード不可：対象年月(B5)が当月と不一致／氏名(B3)が登録名と不一致（簡繁OK）
- ⚠️ 提出は可だが警告：休日（土日・祝日）に勤務時間が入力されている／日付(5行目)が1日〜月末まで揃っていない

## 6. よくある質問

**Q. 社員が送った Excel をこの窓口に転送できますか。**
できません。提出は社員アシスタントが受領します。総務チャネルは管理・集計・催促用です。

**Q. 催促を送ったのに届かない人がいます。**
その人が **総務アシスタントを友だち追加**しているか、LINE の通知設定をご確認ください。提出済の人にはそもそも送られません。

**Q. 自分の名前で認証できません。**
所属部署＋氏名（例：`総務部 高橋 茉鈴`）で入力してください。全角/半角・かなの違いは自動で吸収します。同姓同名のときは末尾に社員番号（例 `S001`）。
