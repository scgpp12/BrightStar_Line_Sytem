# 社員アシスタント（一般社員向け）利用マニュアル

**BrightStar 統合 LINE アシスタント — 社員チャネル**
対象：一般社員（全従業員） / 版本：v1.0 / 最終更新：2026-06-19

> 本チャネルは一般社員向けの **統合フロント** です。**研修の受講・申込** と **勤怠・通勤費の提出** を 1 つの窓口にまとめています。
> 中文：本窗口面向全体员工，把「研修报名」和「考勤·通勤费提交」合并到一个入口。

---

## 1. はじめに（最初の登録 → 毎日のメニュー選択）

### 1-1. 初回登録（最初の1回だけ）
LINE で友だち追加し、**「所属部署 お名前」**を送信します。
```
開発部 山田太郎
```
- 社員名簿と照合し、**在籍者本人**として登録します。
- 同姓同名は社員番号付与：`開発部 山田 E104`
- 名簿に無い場合は人事へご連絡ください。

### 1-2. 毎日の「研修／人事」選択
登録後は、**毎日（東京時間）最初の利用時に「研修」か「人事」を選びます**（ボタンをタップ）。

```
📚 研修 … 研修の受講・お申込み
🗂️ 人事 … 勤怠・通勤費の提出 / テンプレ / 履歴
```

- 選んだモードはその日のあいだ保持されます。
- 日付が変わると、また選択画面が出ます。
- 言語（日本語／中文）も最初に選べます（既定：日本語）。

### 1-3. 登録のやり直し
**「登録解除」** で紐付けを解除し、別の人として登録し直せます。

---

## 2. 研修モードでできること

メニューで **「研修」** を選ぶと、研修の受講者として操作できます。

- 開講中・予定のコース一覧を見る
- コースに **申込／キャンセル** する
- 自分の申込状況・次回のコースを確認する
- 開講の約 1 時間前に **Zoom リンク付きのリマインド**が自動で届く

> 研修の **講師**（コース作成・公開など）は別の「研修アシスタント（講師向け）」をご利用ください。

---

## 3. 人事モードでできること

メニューで **「人事」** を選ぶ、または **Excel ファイルを送る**と、人事モードになります。

- 勤怠・通勤費の **テンプレートをダウンロード**する
- 記入した Excel を **送って提出**する（**再提出も可**＝上書き）
- 自分の **提出履歴**を確認する

### 提出の流れ
1. テンプレートを受け取り、Excel に記入。
2. 記入済みファイルを LINE に**送信** → そのまま提出されます。
3. 締め日までは何度でも送り直せます（最新が有効）。

---

## 4. ヘルプ・キーワード

「ヘルプ」と送ると、機能とキーワードがボタンで表示されます。タップするだけで操作できます。主なキーワードは多言語（日本語・中文・英語・韓国語）に対応しています。

| 操作 | キーワード例 |
|------|------------|
| 研修モードへ | `研修` / `training` / `培训` |
| 人事モードへ | `人事` / `attendance` / `考勤` |
| ヘルプ | `ヘルプ` / `help` / `帮助` |
| 言語切替 | `日本語` / `中文` |
| 登録解除 | `登録解除` |

---

## 5. よくある質問

**Q. 研修と人事、両方使いたい。**
その日のうちに「研修」「人事」と送ればモードを切り替えられます。Excel を送ると自動で人事モードになります。

**Q. 「※Line未登録」と言われます。**
最初に「所属部署 お名前」での登録が必要です。1 つのチャネルで登録すれば、同じ会社の他チャネルとも本人情報は共有されます。

**Q. 提出をやり直したい。**
締め日までは、記入し直した Excel をもう一度送れば上書きされます。

**Q. 別人で登録し直したい。**
`登録解除` → 改めて「所属部署 お名前」を送信。
