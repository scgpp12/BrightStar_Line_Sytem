# 社員アシスタント（一般社員向け）利用マニュアル

**BrightStar 統合 LINE アシスタント — 社員チャネル**
対象：一般社員（全従業員） / 版本：v1.1 / 最終更新：2026-06-26

> 本チャネルは一般社員向けの **統合フロント** です。**研修の受講・申込** と **勤怠・通勤費の提出** を 1 つの窓口にまとめています。
> 中文：本窗口面向全体员工，把「研修报名」和「考勤·通勤费提交」合并到一个入口。

---

## 1. はじめに（最初の登録 → 毎日のメニュー選択）

### 1-1. 初回登録（最初の1回だけ）
LINE で友だち追加し、**「所属部署 お名前」**を送信します。
```
開発部 山田太郎
```
- 社員名簿と照合し、**在籍者本人**として登録します。
- 同姓同名は社員番号付与：`開発部 山田 E104`
- 名簿に無い場合は人事へご連絡ください。

### 1-2. 毎日の「研修／人事」選択
登録後は、**毎日（東京時間）最初の利用時に「研修」か「人事」を選びます**（ボタンをタップ）。

```
📚 研修 … 研修の受講・お申込み
🗂️ 人事 … 勤怠・通勤費の提出 / テンプレ / 履歴
```

- 選んだモードはその日のあいだ保持されます。
- 日付が変わると、また選択画面が出ます。
- 言語（日本語／中文）も最初に選べます（既定：日本語）。

### 1-3. 登録のやり直し
**「登録解除」** で紐付けを解除し、別の人として登録し直せます。

---

## 2. 研修モードでできること

メニューで **「研修」** を選ぶと、研修の受講者として操作できます。

- 開講中・予定のコース一覧を見る
- コースに **申込／キャンセル** する
- 自分の申込状況・次回のコースを確認する
- 開講の約 1 時間前に **Zoom リンク付きのリマインド**が自動で届く

> 研修の **講師**（コース作成・公開など）は別の「研修アシスタント（講師向け）」をご利用ください。

---

## 3. 人事モードでできること

メニューで **「人事」** を選ぶ、または **Excel ファイルを送る**と、人事モードになります。
画面下部の**大きなボタンメニュー**（2026-07 追加）からもワンタップで操作できます：
**勤怠提出／経費提出／その他経費／履歴／研修／人事**。

- 提出用テンプレートを **ダウンロード**する
- 記入した Excel（または **PDF・写真**）を **送って提出**する（**再提出も可**＝上書き）
- **その他経費**（宿泊費など）を用途つきで提出する
- 自分の **提出履歴**を確認する

### 3-1. 提出する書類
| 書類 | 用途 |
|------|------|
| 勤務表 | 勤怠（作業時間） |
| **交通費経費** | 交通費（通勤費）※2026-06 に様式変更。「テンプレ」から最新版を取得してください |
| **その他経費**（2026-07 追加） | 宿泊費・接待費など。用途を入力してからファイルを送信（月内に何件でも可） |

### 3-2. 提出の流れ
1. 「テンプレ」でテンプレートを受け取り、Excel に記入。
2. 記入済みファイルを LINE に**送信** → そのまま提出されます（ファイル名から自動で種別を判定）。
   ボタン「勤怠提出」「経費提出」を押すと、**様式の選択**が出ます：
   - **✅ 正規テンプレ** … 会社の様式。年月・氏名などを自動チェックして受付
   - **📎 その他様式** … 現場様式など。**チェックなし**でそのまま保存（担当者が目視確認）
   ※ 自動チェックで弾かれた場合も「**そのまま提出**」ボタンでチェックなし提出に切り替えられます。
3. 締め日までは何度でも送り直せます（最新が有効）。

> 🆔 保存されるファイル名には**社員番号**が入ります（例：`勤務表（E003_山田太郎）_202607.xlsx`）。同姓同名でも取り違えがありません。

> 📅 **翌月はじめの提出もOK**：月末締めの勤務表・交通費経費が**翌月の頭にずれ込んでも**そのまま提出できます（受付対象＝**前月・当月**分）。保存先とファイル名は**ファイルに記入された年月**どおりに整理されるので、8月に提出した7月分は 7月のフォルダに入ります。それより古い月のファイルは受付できません。

#### PDF・写真（画像）でも提出できます（2026-07 追加）
- 現場様式の PDF やスマホで撮った写真もそのまま送信できます。
- ただし PDF/画像は **年月・氏名などの自動チェックが行えません**。受付後に担当者が目視確認します（受付メッセージにその旨が表示されます）。

#### その他経費の提出（2026-07 追加）
1. ボタン **「その他経費」**（または「その他経費」と送信）
2. 用途を入力（例：宿泊費）
3. 領収書などのファイル（Excel / PDF / 画像）を送信 → 受付完了
- 月内に**複数件**提出できます（上書きされません）。途中でやめる場合は「キャンセル」。

### 3-3. 交通費経費の記入チェック
提出時に内容を自動チェックします。次の不備があると **アップロードできません**（修正のうえ再提出してください）：

- **対象年月（B1）** が提出する月と違う
- **氏名（D6「氏名：◯◯」）** が登録氏名と一致しない
  - ※簡体字／繁体字（例：孫＝孙）はどちらでもOKです
- **出勤状態（D列）に未記入の日がある** … その月の**全ての日**に「出勤／休み／在宅」等を記入してください（未記入の日付がメッセージに表示されます）
- **⑤本人確認のチェックボックスが未チェック** … 内容を確認のうえチェックを入れてください

次は**提出はできますが、リマインド（注意）が出ます**：

- **責任者氏名（D7）が未記入**、または「（本社メンバーの名前）」のまま … **定期券を購入する場合は責任者氏名（承認者）の記入が必須**です
- 提出が成功すると … 「**定期券を購入する際は、定期券の写真添付が必須**」のご案内が出ます

### 3-4. 勤務表（勤怠）の記入チェック
次の不備があると **アップロードできません**（修正のうえ再提出してください）：

- **対象年月（B5）** が提出する月と違う
- **氏名（B3）** が登録氏名と一致しない（※簡体字／繁体字どちらでもOK）

次は**提出はできますが、警告（注意）が出ます**（内容に間違いがなければそのままで結構です）：

- **休日（土日・祝日）に勤務時間が入力されている** … 対象の日付が表示されます
- **日付（5行目）が 1 日〜月末まで揃っていない** … 未入力の日が表示されます

## 4. ヘルプ・キーワード

「ヘルプ」と送ると、機能とキーワードがボタンで表示されます。タップするだけで操作できます。主なキーワードは多言語（日本語・中文・英語・韓国語）に対応しています。

| 操作 | キーワード例 |
|------|------------|
| 研修モードへ | `研修` / `training` / `培训` |
| 人事モードへ | `人事` / `attendance` / `考勤` |
| ヘルプ | `ヘルプ` / `help` / `帮助` |
| 言語切替 | `日本語` / `中文` |
| 登録解除 | `登録解除` |

## 5. よくある質問

**Q. 研修と人事、両方使いたい。**
その日のうちに「研修」「人事」と送ればモードを切り替えられます。Excel を送ると自動で人事モードになります。

**Q. 「※Line未登録」と言われます。**
最初に「所属部署 お名前」での登録が必要です。1 つのチャネルで登録すれば、同じ会社の他チャネルとも本人情報は共有されます。

**Q. 提出をやり直したい。**
締め日までは、記入し直した Excel をもう一度送れば上書きされます。

**Q. 別人で登録し直したい。**
`登録解除` → 改めて「所属部署 お名前」を送信。

**Q. 営業部・総務部・研修など、一般社員以外でも使えますか。**
はい。**部署・役割を問わず**、全社員が本チャネルで登録・提出できます（例：「営業部 山田太郎」）。各職種専用のチャネル（営業／総務 など）とは別に、勤怠・交通費の提出はこの社員アシスタントで行います。

**Q. 未提出のリマインド（催促）はどこに届きますか。**
未提出の場合、**この「社員アシスタント」にリマインド**が届きます（例：「○月の勤務表が未提出です」）。提出済みの方には届きません。
